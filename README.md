# Chinese Restaurant Website

# 1. About the Project
* The project is a Chinese Restaurant Web Application consisting of a React frontend and Node.js,  Express backend connected with MongoDB database.
* Users can create accounts, login securely using JWT authentication, explore menu categories, manage carts, checkout orders, and submit reservations & customer reviews.
* The system includes role based access where Admin users view total users and total revenue, manages the customer orders and update the order status, delete the expired reservations through a dedicated dashboard.

# 2. Aim of the Project

* To design and develop a modern online restaurant management and food ordering system.
* To provide customers with a simple, secure, and responsive platform for ordering food and booking tables online.
* To implement a complete MERN stack application integrating frontend UI, backend APIs, authentication, and database management.

# 3. Purpose of the Project

* To eliminate manual restaurant ordering and reservation processes.
* To improve customer convenience through online menu browsing and digital ordering.
* To help restaurant administrators efficiently manage users, orders, reservations and view total revenues in one system.

# 4. Frontend Implementation

* The frontend of the system is developed using React.js to create a dynamic and responsive user interface.
* Components are designed for Home Page, Menu Display, Cart, Checkout, Order History, Login and Register,  Reservation , Admin Dashboard and Customer Feedback .
* Material UI / CSS styling is used to provide a modern and user-friendly design.
* React Hooks and Context API are implemented for state management, cart handling, and authentication flow.

# 5. Backend Implementation

* The backend is developed using Node.js with the Express.js framework.
* RESTful APIs are created to handle:
      1. User authentication
      2. Menu management
      3. Order processing
      4. Reservation handling
* Middleware is used for request validation, authentication, and error handling.

# 6. Database Implementation

* MongoDB is used as the database to store application data.
* Collections include:
       1. Users
       2. Menu Items
       3. Orders
       4. Reservations
* Mongoose ODM is used for schema creation and database interaction.

# 7. Integration & Testing

* Frontend and backend are connected through APIs.
* API testing is performed using Thunder Client.
* Functional testing ensures smooth cart operations, order placement, and user authentication.
